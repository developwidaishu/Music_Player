# Music_Player

A Simple Music Player that holds the features for the user to 
Play
Pause 
autoplay 
Fast forward the songs and 
turn the volume up or down
using the widgets on the UI. 

Languages used:
HTML5
CSS
JavaScript

APIs used
Fontawesome 4.7
Fontawesome Icons
